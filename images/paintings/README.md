# Paintings Directory

Place your painting images here.

## How to Add Paintings

1. Add your image files to this directory (`images/paintings/`)
2. Edit `paintings.html` and add entries to the `paintings` array:

```javascript
const paintings = [
  { 
    file: "your-image.jpg", 
    title: "Painting Title", 
    artist: "Artist Name", 
    year: 2024 
  },
  // Add more paintings...
];
```

## Supported Formats
- JPG/JPEG
- PNG
- WebP
- GIF
- SVG
